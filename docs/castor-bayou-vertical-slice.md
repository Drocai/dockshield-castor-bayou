# Castor Bayou Vertical Slice

## Title

**DockShield: Castor Bayou Incident**

## Runtime

8-12 minutes for the first playable target.

The earlier 15-25 minute concept is reserved for a later expanded demo.

## Player Fantasy

The player is dropped into a contaminated storm event and must use The Depth's fishing-based hero abilities to rescue civilians, expose spill evidence, and defeat a mutated bayou threat.

## Map

One connected graybox arena:

```text
storm dock -> flooded path -> under-dock tunnel -> fish-processing yard -> boss pool
```

## Mission Flow

1. DockShield weather alert
2. The Reel tutorial: pull/grapple/rescue
3. The Fly tutorial: mark/reveal weak point
4. Lilly tutorial: bind/control hazard
5. Character switching unlock
6. Command Mode v0 unlock
7. Mini boss emerges from toxic water
8. Team finisher moment: Hook, Line & Sinker
9. Extraction and teaser

## Required Prototype Objects

- one grapple point
- one movable object
- one civilian rescue target
- one toxic hazard
- one mutation enemy
- one boss weak point
- one extraction point

## Required Hero Abilities

### The Reel

- Grapple Cast
- Rescue Reel

### The Fly

- Fly Mark
- Compound Vision overlay v0

### Lilly Loch

- Vine Bind
- Healing Moss or hazard control v0

## Required UI

- active hero indicator
- target reticle
- interact prompt
- weather status
- civilian status
- command mode action preview

## Cut From First Slice

- procedural/drop-in map generation
- open world
- full inventory
- gear crafting
- large-scale water simulation
- multiplayer
- full cinematic pipeline
- multiple bosses
- full stealth system
- full XCOM-style squad tactics

## Success Criteria

The slice works if a new player can say:

- The Reel feels like a rescue grappler.
- The Fly changes how I read the battlefield.
- Lilly controls space differently from the others.
- Weather affects decisions.
- DockShield feels different from a normal superhero brawler.
