# $10,000 Prototype Budget

## Budget Goal

The $10,000 budget should produce a fundable/playtestable vertical slice, not a finished commercial game.

## Spend Philosophy

Use free tools wherever possible.

Spend money only where it buys:

- technical implementation speed
- production-quality art or animation
- trailer/pitch quality
- marketplace assets that save weeks
- legal/store/admin clarity

## Budget Allocation

| Category | Budget | Use |
| --- | ---: | --- |
| Unreal/Fab assets | $1,000 | bayou, docks, water props, VFX, UI sounds |
| Gameplay engineer support | $2,000 | targeting, switching, ability architecture |
| Blueprint/C++ cleanup | $1,500 | packaging, source cleanup, build reliability |
| Character/creature art pass | $1,500 | The Reel, The Fly, Lilly, mini boss kitbash/polish |
| Animation/mocap cleanup | $1,000 | rescue, grapple, bind, enemy attacks |
| Audio | $750 | SFX, stingers, temp mix |
| Pitch/trailer/capsule polish | $750 | capture, capsule, deck visuals |
| Store/legal/admin | $500 | Steam/Epic/itch prep, trademark/domain check |
| Contingency | $1,000 | unexpected blockers |

Total: $10,000

## Do Not Spend First On

- broad AI subscriptions
- huge asset libraries
- multiplayer plugins
- procedural world systems
- cinematic tools before gameplay works
- paid code plugins before native Unreal has been tested

## Recommended Paid Help Order

1. Gameplay engineer for targeting and character architecture.
2. Unreal technical artist for graybox-to-readable environment pass.
3. Animator for ability readability.
4. Trailer/capture support after the slice plays well.

## Free/Included Tools First

- Unreal Engine
- Blender
- MetaHuman under standard Unreal license terms
- Poly Haven CC0 assets
- OBS Studio
- Git/Git LFS
- Codex
