## Core Concept

DockShield is a tactical supernatural rescue-action game where mutated heroes defend contaminated waterways, storm zones, marinas, lakes, docks, bayous, and hidden spill sites.

The first playable should prove:

- Pull: The Reel
- Mark: The Fly
- Bind: Lilly Loch
- weather pressure
- civilian rescue
- one mutation enemy
- one mini boss
- command mode v0

## Non-Negotiables

- DockShield is the platform/game title.
- The Depth is the squad.
- Castor Bayou Incident is the first vertical slice.
- Fishing gear logic is central to hero mechanics.
- Weather affects gameplay, not just visuals.
- First build must stay small.
