# Milestone Roadmap

## Goal

Create a fundable/playtestable DockShield vertical slice, not a finished game.

## Milestone 0 - Project Control

Definition of done:

- planning repo exists
- Git LFS configured
- setup docs exist
- backlog exists
- budget exists
- handoff package exists
- local toolchain checker exists

Status: complete.

## Milestone 1 - Toolchain and Unreal Project

Definition of done:

- Unreal Engine 5.7.x installed
- Blender installed
- OBS installed
- Visual Studio C++ workload verified
- private remote repo created
- Unreal Third Person project named `DockShield` created
- project opens cleanly
- initial generated Unreal project committed

Estimated time: 1-2 days.

## Milestone 2 - Targeting Prototype

Definition of done:

- shared target interface exists
- target debug display exists
- one target actor can expose actions by hero
- The Reel can detect and select a target
- one test map proves targeting works

Estimated time: 2-4 days.

## Milestone 3 - First Playable Verb

Definition of done:

- The Reel placeholder character exists
- Pull works on a grapple point
- Pull works on a civilian rescue target
- UI prompt shows valid target action
- interaction can be repeated without breaking state

Estimated time: 3-5 days.

## Milestone 4 - Three-Hero Loop

Definition of done:

- The Reel Pull works
- The Fly Mark works
- Lilly Bind works
- character switching works
- each hero has one target type where they feel different

Estimated time: 1-2 weeks.

## Milestone 5 - Castor Bayou Graybox

Definition of done:

- one connected bayou dock map exists
- mission path is readable
- one civilian rescue exists
- one toxic hazard exists
- one mutation enemy exists
- weather state v0 changes visibility or hazard behavior

Estimated time: 1-2 weeks.

## Milestone 6 - Mini Boss and Command Mode v0

Definition of done:

- mini boss has at least two phases
- weak point can be marked, pulled, and bound
- slow-time command overlay can queue one action
- Hook, Line & Sinker finisher beat is playable or scripted in-engine

Estimated time: 2-3 weeks.

## Milestone 7 - Playtest Build

Definition of done:

- packaged Windows build
- 8-12 minute playable path
- three outside playtesters
- recorded issues
- one 60-90 second capture video

Estimated time: 1 week.
