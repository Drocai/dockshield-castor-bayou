# Playtest Plan

## First Playtest Goal

Find out whether the three-hero fishing logic is understandable and fun.

Do not use the first playtest to judge the whole story, full art direction, or market potential.

## Test Build Requirements

- starts without editor
- 8-12 minute playable path
- clear objective text
- restart option
- no required debug console
- playable with keyboard/mouse first

## Test Questions

Ask each tester:

1. What did The Reel do?
2. What did The Fly do?
3. What did Lilly do?
4. Did weather change what you did?
5. Did the targeting system make sense?
6. Where did you get stuck?
7. Which moment felt most like the actual game?

## Metrics

Track:

- time to first successful Pull
- time to understand target prompt
- failed interactions
- camera confusion
- objective confusion
- combat deaths
- rescue failures
- whether player switches characters without being forced

## Pass Criteria

The build passes if:

- 3 out of 3 testers understand Pull, Mark, Bind
- at least 2 testers use character switching without confusion
- at least 2 testers notice weather or hazard pressure
- no tester gets permanently stuck
